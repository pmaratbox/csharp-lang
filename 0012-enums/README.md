# 0012 — Enums

Define a `Color` enum (`Red`, `Green`, `Blue`) and print the integer value of
`Green` (1) and `Blue` (2). A C# `enum` is a value type backed by an integer
(`int` by default) that auto-numbers its members `0, 1, 2`; the member is not
implicitly an integer, so you cast it with `(int)` to read its numeric value.

## Run

    dotnet run
