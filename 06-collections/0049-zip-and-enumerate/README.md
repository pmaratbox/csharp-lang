# 0049 — Zip & Enumerate

Pair the letters `a, b, c` with the numbers `1, 2, 3` position by position, formatting each pair as `key=value` and printing `a=1 b=2 c=3`. LINQ's `Zip` pairs two sequences with a result selector, stopping at the shorter. Its enumerate counterpart is `Select` with its index overload `(item, i)`.

## Run

    dotnet run
