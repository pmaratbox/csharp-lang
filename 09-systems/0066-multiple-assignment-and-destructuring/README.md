# 0066 — Multiple Assignment & Destructuring

Swap two variables (`a = 1`, `b = 2`) with a single multiple-assignment, then unpack the pair `(3, 4)` into two variables — printing `2 1` then `3 4`. Tuple deconstruction `(a, b) = (b, a)` swaps (the right tuple is built first), and `var (x, y) = (3, 4)` unpacks a tuple.

## Run

    dotnet run
