# 0090 — Binary Search Tree

Insert `5, 3, 8, 1, 4` into a binary search tree and print an in-order traversal (which yields the values in sorted order): `1 3 4 5 8`. A `Node` class with nullable `Left`/`Right` links the tree; `Insert` recurses by comparison and the in-order walk yields sorted values.

## Run

    dotnet run
