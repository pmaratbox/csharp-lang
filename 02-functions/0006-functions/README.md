# 0006 — Functions

Define `Add(a, b)` returning their sum, call it with `3` and `4`, and print
the result. With top-level statements, `Add` is written as a **local
function** declared right alongside the code that calls it; an expression body
(`=> a + b`) is the concise form for a one-line method. String interpolation
(`$"..."`) splices the returned value straight into the output.

## Run

    dotnet run
