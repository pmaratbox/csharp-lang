# 0674 — Node degree

This program uses [QuikGraph](https://www.nuget.org/packages/QuikGraph), C#'s
graph data-structure and algorithms library. It builds the fixed weighted
undirected graph G and queries `AdjacentDegree` to report how many edges are
incident to node `b` (which connects to `a`, `c`, and `d`).

## Run

    dotnet run
