# 0051 — Ternary & Conditional Expressions

For each of `4` and `7`, choose `even` or `odd` with a single conditional expression and print `4: even` and `7: odd`. C#'s `?:` conditional operator picks one of two values of a compatible type. C# also has a ternary-like `??` null-coalescing operator for the null case.

## Run

    dotnet run
