# 0034 — Default & Named Arguments

Give a `greet` function a default greeting, then call it once without the greeting and once overriding it, printing `Hello, Ada` and `Hi, Ada`. C# supports both optional parameters with defaults (`string greeting = "Hello"`) and named arguments (`greeting: "Hi"`). Named arguments may appear out of order as long as the compiler can still bind them.

## Run

    dotnet run
