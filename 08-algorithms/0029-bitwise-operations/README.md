# 0029 — Bitwise Operations

Compute bitwise AND, OR, and XOR on `6` and `3`, plus a left shift of `6` by one bit, printing `and: 2`, `or: 7`, `xor: 5`, and `shift: 12`. C# uses `&`, `|`, `^`, and `<<` on integral types, with `>>` and the unsigned `>>>` right shifts. `6 << 1` doubles to 12, and the operators can be overloaded for user-defined types.

    dotnet run
