# 0056 — Linked List

Build a singly-linked list holding `1`, `2`, and `3`, then traverse it from head to tail and print `1 -> 2 -> 3`. A `Node` class holds `Value` and a nullable `Node? Next` (null at the tail, under `Nullable enable`); walking `node = node.Next` until null traverses the chain.

## Run

    dotnet run
