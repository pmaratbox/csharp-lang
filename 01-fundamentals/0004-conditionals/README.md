# 0004 — Conditionals

Compare `n` against 10 and print whether it's less, equal, or greater. C#
requires parentheses around the condition and braces are conventional even for
a single statement, while `==` on the `int` value type compares values rather
than references. A `bool` prints as `True`/`False` (capitalized), not
`true`/`false`. Edit `n` to `10` or `15` to exercise the other branches.

## Run

    dotnet run
