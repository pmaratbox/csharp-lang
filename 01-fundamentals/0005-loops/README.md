# 0005 — Loops

Print 1..5 with the classic three-part `for` loop. C# also has a `foreach`
form (`foreach (var x in collection)`) for iterating any `IEnumerable`, plus
`while` and `do/while`. A range loop reads naturally as `foreach (var i in
Enumerable.Range(1, 5))`, where the second argument is the count, not the end.

## Run

    dotnet run
